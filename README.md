# Taslow

Taslow es un organizador de tareas que te permite organizar todos tus deberes en una lista de forma sencilla.

## Tecnologías usadas

* HTML
* CSS
* JavaScript

## Deployment

Puedes ver la aplicación en vivo aquí: [Taslow en GitHub Pages](https://mathiasscience.github.io/Taslow/)

## Cómo usar la aplicación

### Añadir tareas
Haz clic en el cuadro que dice `"escribe una tarea"`, escribe tu pendiente y presiona el botón de enviar. ¡Listo! Ya aparecerá en tu lista.

### Terminar tareas
Presiona el botón `"✓"` al lado de cualquier tarea. Esto la moverá automáticamente a la sección de **"tareas hechas"**.

### Eliminar definitivamente
Si quieres borrar algo por completo, ve a la lista de tareas hechas y presiona `"🗑️"`.

### Reabrir tareas
Si terminaste una tarea por error, presiona el botón `"↺"` en la lista de tareas hechas y volverá a la lista principal.

### Cambiar de modo
Presiona el botón debajo de la aplicación que dice `"Cambiar modo"` para alternar entre el modo oscuro y el modo claro.
